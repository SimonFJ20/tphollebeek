0s and 1s

Omtrent da jeg først startede ud med Node og webservere lavede jeg en simpel multiplayer version af Kryds & Bolle som jeg kaldte for 0s And 1s, ud fra at kryds var et 1 og at bolle var et 0.

Desuden var baggrunden også lavet af tekst, for at undgå at skulle importere billeder for spillet.

┌───┬───┬───┐
│ 0 │ 1 │ # │
├───┼───┼───┤
│ 0 │ # │ 1 │
├───┼───┼───┤
│ 0 │ # │ # │
└───┴───┴───┘
PLAYER 0 WON!
- GAME OVER -
